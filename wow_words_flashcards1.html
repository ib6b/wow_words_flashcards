WOW WORDS

Here are some amazing Wow Words you can use in your writing at school to make it more interesting.
If you can add them to your stories your teachers will be really impressed!
Mouse over each word below to see what the word means and some other words you could use instead.
How many Wow Words do you know already?

Abundance

= Lots of

Plenty / Oodles

Accommodate

= to help make something fit

help out / do a favour for / make space for

Acknowledge

= to pay attention to

to notice / to recognise

Adapt

= change to meet the need

alter / change

Advantage

= something that is very helpful

benefit / big help / profit

Aggravate

= to make worse

to annoy / to make more difficult / to upset

Assertive

= confident / saying how you feel about something

speaking up/out / able to express yourself

Assume

= to think you know something

to believe / to suppose / to understand / to guess

Authentic

= the real thing

accurate / actual / genuine / reliable / valid / not fake

Bewildered

= totally surprised and confused

speaking up/out / able to express yourself

Capacity

= the ability to hold or do something

size / how much / ability / skill

Clarify

= to explain

to make something clear / to give a reason

Component

= a part of something

part

Coincidence

= accident / fate

chance

Confine

= to trap something in a limited space

to trap / to imprison / to lock away / to limit to

Conscious

= being aware

alert to / awake / intended to / knowing about / meaning to

Dominate

= to have control over / be the boss of

control / boss around / take over

Disastrous

= awful

horrible / shocking / terrible

Elated

= being extremely happy

excited / joyful / giddy

Hazardous

= extremely dangerous

harmful / risky / unhealthy/ damaging

Intention

= aim / plan

objective / goal / meaning / purpose

Legible

= easy to read

clear / neat

Leisure

= free time

time off / fun / recreation

Literally

= actually, exactly as said

strictly speaking / exactly / in reality

Mutual

= similar

shared / connected / in common

Obstacle

= something that blocks the way

obstrcution / hitch / stumbling block

Perceive

= to understand

to be aware of/ to see / to feel / to grasp

Periodically

= every so often

in cycles / now and again / occasionally

Perturbed

= feeling upset

angered / worried / troubled / annoyed

Ponder

= to think about

to consider/ to mull over / to cogitate / to reflect

Reluctant

= not wanting to

unwilling / hesitant / resistant

Spontaneous

= happening all of a sudden / not planned

spur-of-the-moment / off-the-cuff

Summon

= order to attend / to gather or collect

call for / invite / to muster / to rally

Thrifty

= using money or resources carefully

frugal / economical / parsimonious

Vacant

= empty / unused / unoccupied

blank / void / not filled up

Yield

= to give up or produce

to surrender / relent / blossom / earn
